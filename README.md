## Start

Run the server:

```shell
cargo run --bin server
```

Run the client:

```shell
cargo run --bin client
```

## Prerequisites

Debian:

```shell
apt update && apt upgrade -y
apt install -y protobuf-compiler libprotobuf-dev
```

More info for [tonic](https://github.com/hyperium/tonic#dependencies).
